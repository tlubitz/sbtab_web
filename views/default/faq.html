Frequently asked questions

Are SBtab files spreadsheet or text files?

SBtab tables can be stored both as spreadsheet (.xls) or delimited text (.csv, .tsv, .tab) files. However, SBtab refers only to the plain content of the table, so no macros or references (like in Excel) can be used. Like spreadsheet files, SBtab documents can consist of several tables. In delimited text formats, these can either be stored as separate files, or merged into one table file.

Are there any restrictions on filenames or filename extensions like ".tsv" or ".xls"?

The SBtab format as such does not impose any restrictions on filenames, nor does it require a specific filename extension. SBtab files stored as excel sheets, for instance, will have the extension ".xls". The SBtab online tools (and the python programs behind it), by default, use the extensions ".tsv" and ".csv", respectively, for tab-delimited and comma-delimited text files. When an SBtab document is exported to several delimited text files, the filenames will be chosen according to the scheme "[SBTAB DOCUMENT NAME]_[TABLE TYPE].tsv" or, in case of ambiguities "[SBTAB DOCUMENT NAME]_[TABLE TYPE]_[TABLE NAME].tsv".

Regarding filename extensions, the python implementation of SBtab supports comma-separated and tab-separated tables, as well as excel spreadsheet files. By default, the python code exports tab-separated files and uses the filename extension ".tsv". In case of conflicts, users may have to simply rename their files. When importing a table, the code tries to guess whether commas or tabs are used as delimiters. When using commas as delimitersm , users have to make sure that no commas are used elsewhere in the table (or that all elements are given in double quotes).

How can I ..

  1. .. convert an existing data table into SBtab? Usually, relatively little changes are necessary. A short description is given here. For more details, consult the SBtab draft specification.

  2. .. write and edit SBtab files? You can use any spreadsheet program or text editor. For the correct SBtab syntax, please have a look at our example files and read the SBtab draft specification.

  3. .. convert biochemical models between the SBtab and SBML formats? To convert network models between SBtab and SBML, you can either use the online converter or our python code.

  4. .. check SBtab files for syntactic correctness? To check SBtab files for correctness, you can either use the online validator or our python code.

  5. .. customize the SBtab format for your own data types? To customise the SBtab format for your needs, you need to provide a new Definition table defining the allowed table types and column names. To learn about the syntax, have a look at the existing definition table definitions.tsv. For usage within the online tools, you can upload your modified definition table at definitions.tsv. Please note that, when using a new Definition table, the predefined Definition table will be completely overridden, so any tables and columns to be used (also the predefined ones) must be listed in the new table.
