SBtab Tools¶

These functions facilitate the use of SBtab. They can be used to create SBtab objects, by merging strings or read files, respectively.

SBtabTools.oneOrMany(spreadsheet_file)[source]¶

Checks for multiple tables in a file and cuts them into separate tablib object.

Parameters:

spreadsheet_file : tablib object

Tablib object of the whole SBtab table.
SBtabTools.openSBtab(filepath)[source]¶

Opens SBtab from file.

Parameters:

filepath : str

Path of the spreadsheet file.
SBtabTools.createDataset(header_row, columns, value_rows, filename)[source]¶

Creates an SBtab object by merging strings or list of strings. Takes a header row, main column row, and the value rows as lists of strings and returns an SBtab object.

Parameters:

header_row : str

String of the header row.

columns: list

List of strings, names of the columns.

value_rows : list

List of lists containing the different rows of the table.
