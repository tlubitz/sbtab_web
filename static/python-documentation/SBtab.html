SBtab¶

SBtab is a uniforming table format and designed for the use in Systems Biology. Furthermore, it is a useful format to import stored information into Python objects to process and manipulate it.

How to load tablib object: Use “SBtabTools.openSBtab(filepath)” to create SBtab Python object. Attention: Only ‘tsv’, ‘csv’, ‘tab’ and ‘xls’ are supported.

See specification for further information.
